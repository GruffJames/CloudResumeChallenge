Work

2019-Present

Clinisys MIPS, Software Developer-Microbiology coordinator

Member of a development team for the GLIMS product, which is a laboratory information management system (LIMS) used as a critical piece of laboratory automation infrastructure for 200+ hospitals and laboratory networks across europe. The role was initially based in Gent, Belgium before I moved to Glasgow to continue working within a team there. In March 2024 I received the “Practical thinker” company award for my performance and contributions within my team.

Responsibilities include:

  • Collaborating with an international team of developers, testers, and product analysts to develop new features as well as maintain existing features, to ensure scalability, security, and optimal performance.
  • Work with product and UX designers to deliver intuitive and user-friendly interface using OpenEdge Ultra Controls for Microsoft .NET framework.
  • Developing relational database schema.
  • Review colleagues' code and advise on improvements.
  • Create functions for the in-house written scripting language MISPL.
  • Create unit tests to ensure future changes do not break current required functionality as well as to create new features by test-driven development (TDD).
  • Investigate and debug issues escalated to the development team and provide technical analysis for corrections either within the code or by customer configuration.
  • If necessary, provide hotfixes for customers generated using Jenkins builds.
  • This involves reading client/server log files for information, reading documentation of the product, analysing GIT logs to understand the context of past code changes,discussing the issue with customer support from the local country, and combining all that with learned knowledge of the product.
  • The specialist role of microbiology coordinator, where I am the primary contact for incidents relating to that module, as well as completing the majority of related development work.
  • Work on large projects transitioning from initial designs to production in a matter of weeks.

Large projects completed

Alles gurgelt”/”Everything gurgles

Worked as part of a team to create the "Alles gurgelt”/”Everything gurgles” COVID platform for Vienna in Austria. The project created a platform to give people in the region access to weekly COVID lab tests. To ensure the project was able to reach its eventual 500,000 samples a day, It required a new specimen pooling module to provide a large increase in performance to enable the massive throughput of samples required.

This involved creating:

  • A screen for specimen entry via barcodes, a query browser for viewing, processing, and generating pooling statistics as well as a number of other screens for displaying other information using OpenEdge Ultra Controls for Microsoft .NET framework.
  • Add functions to the in-house scripting language MISPL.
  • Database schema and backend code to handle the above features.
