Website architecture

infrastructure as code - Terraform

The website is hosted on AWS with almost all of the configurations stored and uploaded with Terraform

Website pages

For the website front end I used a free template from ©Design: HTML5 UP. I then modified it for my content and any some tweaks here and there.
I decided against writing my own HTML/CSS as that would dramatically increase the scope of the project, and I'm was interested in learning cloud infrastructure skills required here.
